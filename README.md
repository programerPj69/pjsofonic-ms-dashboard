
# PjSofonic MS - Integrated Management System

PjSofonic MS ek comprehensive **Enterprise Resource Planning (ERP)** solution hai jo software development companies ke liye banaya gaya hai. Ye project employee management, project tracking, CRM, aur collaboration ko ek hi platform par lata hai.

## 🚀 Key Features

* **HR & Employee Management:** Attendance tracking, payroll generation, aur leave management.
* **Project & Production:** Task management (Kanban Board), development pipeline, aur Quality Assurance (QA) tracking.
* **Customer Relationship (CRM):** Client requirements management aur complaint ticketing system.
* **Collaboration Suite:** Real-time internal chat system aur meeting/event scheduler.
* **Role-Based Access (RBAC):** Admin, Developer, Support Team, aur Client ke liye alag access control.

## 🛠 Tech Stack

* **Frontend:** React.js, Tailwind CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB / MySQL
* **Real-time:** Socket.io (Chat & Notifications)
* **Authentication:** JWT (JSON Web Tokens)
