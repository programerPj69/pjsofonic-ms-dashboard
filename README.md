
# PjSofonic MS - Integrated Management System
PjSofonic MS is a comprehensive Enterprise Resource Planning (ERP) solution designed specifically for software development companies. It consolidates employee management, project tracking, CRM, and internal collaboration into a single, efficient platform...

## 🚀 Key Features

* **HR & Employee Management:** Attendance tracking, payroll generation, aur leave management.
* **Project & Production:** Task management (Kanban Board), development pipeline, aur Quality Assurance (QA) tracking.
* **Customer Relationship (CRM):** Client requirements management aur complaint ticketing system.
* **Collaboration Suite:** Real-time internal chat system aur meeting/event scheduler.
* **Role-Based Access (RBAC):** Admin, Developer, Support Team, aur Client ke liye alag access control.

## 🛠 Tech Stack

* **Frontend:** React.js, Tailwind CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB / MySQL
* **Real-time:** Socket.io (Chat & Notifications)
* **Authentication:** JWT (JSON Web Tokens)

## 🏗 System ArchitectureControllers: •Handles business logic for every module.
 •Models: Defines database schemas and data structures.
 •Routes: Manages API endpoints (e.g., ⁠/chat⁠, ⁠/tasks⁠, ⁠/salary⁠).
 •Views: Frontend components and user interface files.
 •Config: Database connections and environment variables.


## 📂 Folder Structure

```text
/PjSofonic-MS
├── /controllers    # Business logic
├── /models         # Database schema
├── /routes         # API Endpoints
├── /views          # Frontend components
├── /public         # Static assets
└── /config         # Environment configurations

```
